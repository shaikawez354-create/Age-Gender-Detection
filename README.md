# Age and Gender Detection using OpenCV and Flask

This project detects human age and gender in real-time using a webcam.

## Technologies Used
- Python
- OpenCV
- Flask
- Deep Learning

## Features
- Real-time face detection
- Age prediction
- Gender prediction
- Live webcam streaming using Flask

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Run the project
python main.py

3. Open browser
http://127.0.0.1:5000
